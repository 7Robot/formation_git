Commandes de gestion locale :


git init --> initialise un dépôt git local

git branch --> affiche les branches locales
git branch Nom_branche_locale --> crée la branche locale Nom_branche_locale
git branch -d Nom_branche_locale --> supprime la branche locale Nom_branche_locale

git checkout Nom_branche_locale --> bascule sur la branche locale Nom_branche_locale
git checkout -b Fin_commande --> Equivaut à la commande "git branch Fin_commande", et bascule sur la branche en question

git merge Nom_branche_locale --> importe les modifications de la branche locale Nom_branche_locale sur la branche actuelle

git status --> montre les fichiers modifés
git add . --> ajoute toutes les modifications locales au commit
git commit -m "Message" --> crée le commit local avec les fichiers modifiés avec comme descritpion "Message"

git stash --> enregistre les modifications, mais ne les applique pas (remisage)
git stash list --> liste les remisages
git stash apply Nom_remisage --> réapplique le remisage Nom_remisage

git reset HEAD --> annule tous les changements depuis la dernière synchronisation avec le dépôt distant
git reset HEAD Nom_fichier --> annule la modification sur le fichier Nom_fichier depuis la dernière synchronisation avec le dépôt distant



Commandes de gestion distante :


git clone URL Répertoire_cible --> clone le dépôt distant à l'adresse URL vers le répertoire cible Répertoire_cible
git remote add origin https://github.com/Nom_utilisateur/Nom_dépôt_distant.git --> configure le dépôt local pour suivre le dépôt distant à l'adresse https://github.com/Nom_utilisateur/Nom_dépôt_distant.git

git fetch --> synchronise les dépôts, à faire avant chaque opération qui concerne le dépôt distant
git merge --> fusionne la branche distante sur la branche locale actuelle

git branch -r --> affiche les branches distantes
git branch Nom_branche_locale origin/Nom_branche_distante --> crée la branche locale Nom_branche_locale qui est une copie de la branche distante Nom_branche_distante

git push --> envoie les modifications locales sur la branche distante
git push --set-upstream origin Nom_branche_distante --> configure la branche distante Nom_branche_distante pour suivre la branche locale actuelle
git push origin :Nom_branche --> supprime la branche distante

git rm --cached nom_fichier --> supprime l'index du fichier nom_fichier
